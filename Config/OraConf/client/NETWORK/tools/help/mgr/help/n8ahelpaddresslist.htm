ADDRESS_LIST

A list of addresses that share a set of common characteristics. In the following example, ADDRESS_LIST lists two address with client load balancing turned on:

(ADDRESS_LIST=

(LOAD_BALANCE=on)

(ADDRESS=

(PROTOCOL=tcp

(HOST=sales-pc)

(PORT=1521))

(ADDRESS=

(PROTOCOL=tcp)

(HOST=hr-server)

(PORT=1521)))
