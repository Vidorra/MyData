一般参数：一般信息

一般信息选项卡使您能够配置监听程序的一般参数。

监听程序名

显示监听程序的名称。

启动等待时间

指定在响应监听程序控制实用程序 START 命令之前监听程序等待的时间 (秒)。

注：此设置已过时，在以后的发行版中不再支持。如果运行监听程序时需要此参数，请通知 Oracle 技术支持。

选项

在关闭时保存配置

选择此选项以永久保存由监听程序控制实用程序 SET 命令所作的任何更改。

SNMP 联系信息

输入任意文本，此文本将写入 snmp.ora 文件供 Oracle Enterprise Manager 使用。此信息可通过任何 SNMP 连接发送，并且可以是任意字符串。它不会影响创建 SNMP 连接的进程。此信息一般由客户机的用户或管理员的联系信息组成。
