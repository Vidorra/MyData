“文件”菜单

文件菜单提供打开、还原、保存和另存为等操作。

打开网络配置

选择此项以打开本地配置文件。

保存网络配置

选择此项以将当前配置设置保存到默认位置。

另存为

选择此项以将当前配置设置保存到非默认位置。

还原为已保存的配置

选择此项以不保存任何当前配置设置。而是恢复最后一次保存的配置。

退出

选择此项以关闭 Oracle Net Manager。
