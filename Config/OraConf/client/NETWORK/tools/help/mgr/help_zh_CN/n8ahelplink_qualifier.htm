链接限定词

附加在全局数据库链接后面，为数据库用户名和口令身份证明提供替代设置的限定词。例如，可以将链接限定词 emp 附加在全局数据库链接 sales.us.example.com 的后面。到具有全局数据库链接和链接限定词的到远程数据库的连接将是：

CONNECT scott/tiger@sales.example.com@emp
