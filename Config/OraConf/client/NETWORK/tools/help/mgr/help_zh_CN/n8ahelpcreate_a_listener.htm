创建监听程序

为了数据库接收连接，监听程序必须存在于服务器上。

注：在安装过程中，创建了名为 LISTENER 的默认监听程序。

要创建监听程序，请执行以下操作：

  1. a. 在导航器窗格中，展开 Oracle 网络配置 > 本地 > 监听程序。

  2. 在工具栏中选择加号 (+) 或选择编辑 > 创建。将显示选择监听程序名对话框。

  3. 接受监听程序名字段中提供的监听程序名，或用另一个名称编辑该字段。第一个监听程序的默认监听程序名是 LISTENER。后面的监听程序名默认为 LISTENERn。在任何给定的节点中监听程序名必须是唯一的。

  4. 选择确定。

  5. 完成配置监听程序地址中的步骤 4-6。
