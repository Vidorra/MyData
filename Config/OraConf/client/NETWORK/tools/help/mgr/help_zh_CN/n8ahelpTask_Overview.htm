任务概览

以下是说明如何为网络配置 Oracle 高级安全性的一系列步骤：

  1. 选择认证方法。

  2. 区分在客户机/服务器协商过程中将使用的认证方法的优先次序。

  3. 为选定的每种认证方法配置其他参数。

  4. 配置客户机端和服务器端的完整性 (校验和级)。

  5. 配置客户机端和服务器端加密。

  6. 配置客户机端和服务器端的安全套接字层 (SSL) 通讯设置。
