配置 Oracle 高级安全性认证方法

集中的、安全的验证方法使您高度信任分布式环境中的用户、客户机和服务器的身份。网络验证方法还为用户提供了一次登入的好处。

使用 Oracle Net Manager 配置下列验证适配器：

  • Kerberos

  • CyberSafe

  • RADIUS

  • SSL

  • Windows NT Native 认证

注：如果配置了使用 SSL 的 TCP/IP 协议，在默认情况下将配置安全套接字层 (SSL)。在此选项卡中选择的任何验证方法都将覆盖 SSL 的验证功能。使用 SSL 选项卡配置其他的 SSL 选项。如果您正在使用带有 SSL 的 TCP/IP 协议，请不要使用任何其他验证方法。

要配置验证方法，请执行以下操作：

  1. 在导航器窗格中，展开 Oracle 网络配置 > 本地 > 概要文件。

  2. 从右侧窗格的列表中，选择 Oracle Advanced Security。

  3. 选择验证选项卡。

  4. 从可用方法列表中选择您要使用的验证方法，然后选择右箭头按钮 (>)。

  5. 根据您要使用的方法为验证方法排序。在所选方法列表中选择验证方法，并使用上移或下移按钮使所选方法在列表中上移或下移。

  6. 选择其他参数选项卡，为以下验证方法配置其他参数：

    • Kerberos

    • CyberSAFE

    • RADIUS

  8. 选择文件 > 保存网络配置。
