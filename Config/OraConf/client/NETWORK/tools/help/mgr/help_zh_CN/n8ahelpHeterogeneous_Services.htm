异构服务

一种集成组件，它提供用于从 Oracle 数据库服务器访问非 Oracle 系统的一般技术。异构服务使您能够进行以下操作：

  • 使用 Oracle SQL 可透明地访问存储在非 Oracle 系统中的数据，就好象数据驻留在 Oracle 服务器中一样。

  • 使用 Oracle 过程调用从 Oracle 分布式环境透明地访问非 Oracle 系统、服务或应用程序编程接口 (API)。

要与非 Oracle 系统建立连接，Oracle8i 数据库会通过网关上的监听程序启动一个代理进程。为使 Oracle8i 数据库连接到这个代理，服务器必须配置有网络服务名以访问代理，而且监听程序必须配置有该代理的服务信息。
