验证方法

使您能够高度信任分布式环境中的用户、客户机和服务器的身份的安全方法。网络验证方法还为用户提供了一次登入的好处。根据是否安装了 Oracle Advanced Security，Oracle9i 和 Oracle8i 中支持以下验证方法：

  • CyberSafe

  • KERBEROS(v5)

  • RADIUS

  • SSL

  • Windows NT Native 认证
