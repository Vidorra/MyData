本地命名配置概览

“管理员”安装或“运行时”安装

在“管理员”或“运行时”客户机安装过程中，如果尚未配置目录用法，Oracle Net Configuration Assistant 将提示您在 tnsnames.ora 文件中配置网络服务名。

定制安装

如果在客户机或服务器定制安装期间选择本地命名作为命名方法，Oracle Net Configuration Assistant 将提示您在 tnsnames.ora 文件中配置网络服务名。

具有 Oracle Net Manager 的配置

Oracle Net Manager 使您能够配置网络服务名以及：

  • 创建非 Oracle 数据库服务的网络服务名

  • 通过多个协议地址配置到监听程序的多个路由

  • 配置数据库服务或网络服务名的高级 CONNECT_DATA 信息。

要访问本地命名配置，请执行以下操作：

从导航器窗格中选择本地 > 服务命名文件夹，以便创建网络服务名或修改连接描述符信息。

有关详细信息，请参阅 Oracle10i Net Administrator's Guide 中的第 8 章“Configuring Naming Methods”。
