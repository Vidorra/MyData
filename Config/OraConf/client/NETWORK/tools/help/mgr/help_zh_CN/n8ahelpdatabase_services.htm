数据库服务

数据库服务选项卡使您能够指定监听程序必须接收连接请求的数据库服务。监听程序需要该信息以将客户机连接到数据库。

Oracle9i 和 Oracle8i 实例使用动态服务注册来通知监听程序有关其数据库服务的信息。Oracle7 和 Oracle8 发行版 8.0 数据库以及 Oracle Enterprise Manager 工具要求 listener.ora 文件中的静态服务配置。

选择添加数据库以添加数据库服务信息。

全局数据库名

输入根据初始化参数文件中的 SERVICE_NAMES 参数值或 DB_NAME 和 DB_DOMAIN 参数值提供的数据库名和域名。

Oracle 主目录

输入数据库的 Oracle 主目录位置。

注：Windows NT 已经知道 Oracle 主目录。因此，连接 Windows NT 上的服务不需要此字段。

SID

输入实例的 Oracle 系统标识符 (SID)。

“添加数据库”按钮

选择此按钮为另一个数据库配置服务信息。此时将显示新的数据库选项卡，可以在该选项卡中输入另一个数据库的服务信息。

删除数据库按钮

选择此按钮以删除所选的数据库选项卡及其服务信息。
