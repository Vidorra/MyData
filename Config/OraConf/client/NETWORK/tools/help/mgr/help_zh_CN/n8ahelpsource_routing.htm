源路由选择

指定使用所有网络路由。源路由通常用于有很多连接需要访问相同的服务的环境。Oracle Net 提供通过 Oracle Connection Manager 进行路由选择。

有关 Oracle Connection Manager 的信息，请参阅 Oracle10i Net Administrator's Guide 中的第 1 章“Networking Challenges in the Internet Age”。
