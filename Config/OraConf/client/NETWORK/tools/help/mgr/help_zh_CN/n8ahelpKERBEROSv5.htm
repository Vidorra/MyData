KERBEROS(v5)

KERBEROS(V5) 验证方法要求您设置以下参数：

元素

说明
服务

输入验证服务的名称：KERBEROS。

身份证明高速缓存文件

 Kerberos 身份证明高速缓存文件的完整路径：默认值与操作系统相关。
配置文件

Kerberos 配置文件的完整路径。此文件包含默认密钥分发中心 (KDC) 的领域并将主机映射到 KDC 主机：默认值与操作系统相关。
领域转换文件 Kerberos 领域转换文件的完整路径。此文件将主机名或域名映射到领域：默认值与操作系统相关。
键表 Kerberos 主用户 (组)/密钥映射文件的完整路径。Oracle 服务器使用此路径来提取密钥，并对从客户机传入的验证信息进行解密。默认值与操作系统相关。
时钟偏差 在该时间后 Kerberos 身份证明被认为过期的时间 (秒)。默认值为 300 秒
