监听程序控制实用程序

包含在 Oracle Net Services 中的实用程序，用来控制各种监听程序功能，如启动、停止以及获取监听程序的状态。监听程序控制实用程序以下列方式执行：

lsnrctl command [listener_name]

[listener_name] 是在 listener.ora 文件中定义的监听程序名。如果正在使用默认的监听程序名 LISTENER，则不必标识监听程序。

有关详细信息，请参阅 Oracle10i Net Services Reference Guide 中的第 1 章“Listener Control Utility”。
