CONNECT_DATA

连接描述符中定义目标数据库服务名或 Oracle 系统标识符 (SID) 的部分。在以下示例中，SERVICE_NAME 指定了一项名为 sales.us.example.com 的数据库服务：

(DESCRIPTION=

(ADDRESS=(PROTOCOL=tcp)(HOST=sales-pc)(PORT=1521)

(CONNECT_DATA=(SERVICE_NAME=sales.us.example.com)))

可使用高级连接信息配置 CONNECT_DATA。
