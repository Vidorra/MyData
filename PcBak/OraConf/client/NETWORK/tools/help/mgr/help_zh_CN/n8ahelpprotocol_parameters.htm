协议参数

用协议地址来标识监听程序、Oracle Names Server 和 Oracle Connection Manager。需要使用这些服务的客户机也必须配置同样的地址。下表说明由支持的 Oracle 协议使用的参数：

协议

 参数

说明
IPC PROTOCOL

指定 ipc 作为值。

 

 KEY 指定服务的唯一名称。Oracle Corporation 建议使用服务的服务名或 Oracle 系统标识符 (SID)。
命名管道 PROTOCOL

指定 nmp 作为值。
  SERVER 指定 Oracle 服务器所在计算机的名称。
  PIPE 指定用来连接数据库服务器的管道名称。此名称可以是任何名称。
TCP/IP 和使用 SSL 的 TCP/IP PROTOCOL 指定 tcp 作为 TCP/IP 的值，或指定 tcps 作为使用 SSL 的 TCP/IP 的值。
  HOST 指定计算机的主机名或 IP 地址。
  PORT 指定监听端口号。
