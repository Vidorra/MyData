区分命名方法的优先次序

当配置好命名方法后，必须在概要文件中区分它们的优先次序。列表顶部的命名方法首选被使用，用于将网络服务名或数据库服务名解析为连接描述符。如果此方法不能解析名称，将使用列表中的第二个方法。

要指定命名方法的顺序，请执行以下操作：

  1. 在导航器窗格中，展开 Oracle 网络配置 > 本地 > 概要文件。

  2. 在右窗格的列表中，选择命名。

  3. 选择方法选项卡。

  4. 从可用方法列表中选择命名方法，并选择右箭头按钮。选定的命名方法就移动到选定方法列表中。

  5. 根据在解析网络服务名或数据库服务名时 Oracle Net 使用命名方法的先后顺序对其进行排序。从所选方法列表中选择命名方法，然后选择上移或下移上下移动列表中的选项。

  6. 选择文件 > 保存网络配置。
