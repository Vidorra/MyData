Microsoft Active Directory

包含在 Windows 2000 Server 中符合 LDAP 的目录服务。它存储网络上对象的有关信息，并且使用户和网络管理员可以使用这些信息。“活动目录”还提供使用单一登录进程对网上资源的用户访问。

可将“活动目录”配置为存储客户机可以访问的服务信息的目录命名方法。
