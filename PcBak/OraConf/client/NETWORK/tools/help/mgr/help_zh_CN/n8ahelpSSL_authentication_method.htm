SSL

一种使用安全套接字层 (SSL) 的验证方法。SSL 是由 Netscape Communications Corporation 为保护网络连接而开发的工业标准协议。SSL 使用公共密钥基础结构 (PKI) 提供验证、加密和数据完整性。

有关详细信息，请参阅 Oracle Advanced Security Administrator's Guide。
