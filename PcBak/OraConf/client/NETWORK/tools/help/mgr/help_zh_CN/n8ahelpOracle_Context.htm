Oracle Context

位于命名上下文或未发布的目录条目下的目录信息树 (DIT) 中的 cn=OracleContext 的相对唯一判别名。Oracle Context 包含用于 Oracle 软件的条目，例如用于目录命名的数据库服务和网络服务名。一个目录中可以有一个或多个 Oracle Context。

Oracle Internet Directory 自动在 DIT 结构的根目录中创建 Oracle Context。根 Oracle Context 的唯一判别名 (DN) 为(dn:cn=OracleContext)。

Oracle Net Manager 启动时，使用由 Oracle Net Configuration Assistant 配置的默认 Oracle Context。您可以将此 Oracle Context 更改为另一个环境。对于大多数部署来讲，需要创建附加的 Oracle Context。
