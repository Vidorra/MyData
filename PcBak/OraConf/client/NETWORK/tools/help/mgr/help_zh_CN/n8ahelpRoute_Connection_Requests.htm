路由连接请求

可以配置客户机以使连接请求总是由专用服务器进程或者 Oracle Connection Manager 负责。

要路由连接请求，请执行以下操作：

  1. 在导航器窗格中，展开 Oracle 网络配置 > 本地 > 概要文件。

  2. 从右侧窗格的列表中选择一般信息。

  3. 选择路由选择选项卡。

  4. 选择希望路由连接请求的首选方式。

  5. 选择文件 > 保存网络配置。
