Oracle Internet Directory

作为 Oracle 数据库的应用程序而实现的目录服务。它能检索分散的用户和网络资源的信息。它将轻型目录访问协议 (LDAP) 第 3 版 (开放式 Internet 标准目录访问协议) 与 Oracle 数据库的高性能、可伸缩性、健壮性和可用性结合起来。

有关详细信息，请参阅 Oracle Internet Directory Administrator's Guide。
