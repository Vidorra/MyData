配置外部命名方法

您可以配置外部命名方法所必需的客户机参数，包括概要文件中的网络信息服务 (NIS)。

要配置外部命名方法参数，请执行以下操作：

  1. 在导航器窗格中，展开 Oracle 网络配置 > 本地 > 概要文件。

  2. 在右窗格的列表中，选择命名。

  3. 选择外部选项卡。

  4. 在相应的字段中为您正在使用的外部命名方法输入一个值。

  5. 选择文件 > 保存网络配置。
